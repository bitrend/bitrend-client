# Commit Convention
| Type         | 설명                                           |
| ------------ | -------------------------------------------- |
| **feat**     | 새로운 기능 추가                                    |
| **fix**      | 버그 수정                                        |
| **docs**     | 문서 수정 (README, 주석 등)                         |
| **style**    | 코드 스타일 변경 (포맷팅, 세미콜론 누락 등 비즈니스 로직에 영향 없는 변경) |
| **refactor** | 코드 리팩토링 (성능 개선, 구조 변경 등)                     |
| **test**     | 테스트 코드 추가/수정                                 |
| **chore**    | 빌드 설정, 패키지 매니저 설정, 기타 잡일                     |
| **perf**     | 성능 개선                                        |
| **ci**       | CI/CD 관련 설정 변경                               |

**⚠️ 주의 앞으로 모든 커밋시에는 스코프를 지정해주세요!!**
EX) "feat(dashboard)::그래프 UI 에니매이션 추가"
